1. META TAGS -

1.1 CHARSET - to set character set of our page.

1.2 DESCRIPTION - to set description of our webpage so that it will appear as a snippet when user share our page on social media.

1.3 KEYWORDS - to provide information about our website content in very short words

1.4 AUTHOR - to provide information about owner of website

1.5 VIEWPORT - set our webpage width to device width and set the initial zoom level to 100%
